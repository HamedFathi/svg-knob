# svg-knob

A flexible and customizable knob for your web applications.

## Usage

    .knob {
        width: 100px;
    }

    <svg class="knob" id="knob"></svg>

    <script type="module">
        import knob from './svg-knob.js';
        var k = new knob(document.getElementById('knob'), { /* config... */ });        
    </script>
